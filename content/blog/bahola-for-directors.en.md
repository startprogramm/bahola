# Know How Every Class Is Doing — Right Now

School directors usually have to wait. Wait for end-of-term reports. Wait for teachers to build spreadsheets. Wait for problems to show up in inspections — when it's already too late to fix them.

Bahola changes that. Directors get **a live view** of what's happening in every classroom, every week.

## What You Can See

When you log in as a director, you see a school-wide view that answers three key questions:

### How are students doing?
- Average scores across all subjects and classes
- How students are spread out — how many are above 80%, below 50%
- Trends over time — is performance going up or down?

![Grades overview — every student, every test, every score in one table](/blog/app-grades-tab.png)

### Are teachers running tests?
- Which teachers are using AI grading regularly
- How often each class is tested
- Classes where no tests have been run recently

### Where do we need to step in?
- Classes or subjects with weak results
- Students who may need extra help based on their score trends
- Topics that students across many classes are getting wrong

## Stay on Top of Things in Real Time

![Bahola dashboard — teacher and class activity at a glance](/blog/cover-dashboard.jpg)

Old-school school management means checking in once a term. Bahola gives you current data all the time. If a class starts struggling in week four, you know in week four — not after final exams.

With current data, you can:

- **Step in early** — give extra help before small problems grow
- **Spot great teaching** — see which teachers and methods get the best results
- **Use resources wisely** — know where training is most needed
- **Report with confidence** — show accurate, up-to-date data to leadership and inspectors

> "I used to have to ask teachers for updates. Now I can see exactly how every class is doing before I walk into a meeting." — Director, Presidential School

## Built for Presidential School Standards

Presidential Schools have high standards. Directors are responsible for outcomes at every level.

Bahola was built with that in mind. The director dashboard matches your school's subject layout. You can start with school-wide averages and drill down to a specific class, a specific student, and a question-by-question score breakdown.

No vague numbers that hide problems. The details are always one click away.

## Getting Started as a Director

Setup is fast:

1. **Register with the Director role** and set up your school profile
2. **Invite your teachers** — they get a link, create accounts, and start their classes
3. **Teachers run tests** — data comes in automatically with each submission
4. **Check the director dashboard** — review trends weekly

No downloads or IT setup needed. Bahola runs in any browser, on any device.
