# Reel 76 — Parent Sees the Grade (45 Seconds)

## Concept
Told entirely from a parent's POV. They receive a link from the teacher, tap it on their phone, and see a full AI-explained result — no login, no app, no confusion. Warm, relatable, emotionally resonant. Ends with the parent nodding, satisfied. Targets teachers who want to show parents why TezTekshir builds family trust.

## Hook (0–5s)
Close-up of a phone screen. Notification-style animation: "O'qituvchi sizga baho natijasini ulashdi."
Parent thumb taps the link.
Bold overlay: **"Ota-ona yangi SMS olmadi. U haqiqiy natijani ko'rdi."**

## Scene Breakdown

**[0–7s] — Muammo / Проблема**
- Parent at kitchen table, phone in hand
- Overlay (UZ): "Har doim o'qituvchiga qo'ng'iroq qilish kerakmi?"
- Overlay (RU): "Всегда нужно звонить учителю?"
- Quick flash of missed calls, unanswered messages — familiar chaos

**[7–18s] — Havola keldi / Ссылка пришла**
- Teacher hand on phone — tap "Ulash / Поделиться"
- Parent phone vibrates
- Parent opens link — TezTekshir result page loads instantly
- Screen animation shows: student name (blurred), score "18/20", AI feedback breakdown
- Overlay (UZ): "Baho: 18/20 ✅"
- Overlay (UZ): "AI izohi: nima to'g'ri, nima yo'q"
- Overlay (RU): "Объяснение: что правильно, где ошибка"

**[18–30s] — Ota-ona tushunadi / Родитель понимает**
- Parent reads feedback — face shifts from concern to understanding
- Overlay (UZ): "Faqat raqam emas — sabab ham bor"
- Overlay (RU): "Не просто цифра — есть объяснение"
- Close-up: specific AI feedback text on screen (blurred/anonymized student name)
- Parent nods — no confusion, no anxiety

**[30–40s] — Shaffoflik / Прозрачность**
- Split screen: parent left, teacher right — both calm
- Overlay (UZ): "O'qituvchi: bir marta ulashdi."
- Overlay (RU): "Учитель: поделился один раз."
- Overlay (UZ): "Ota-ona: hamma narsani tushundi."
- Overlay (RU): "Родитель: понял всё."
- No phone call. No back-and-forth. Just trust.

**[40–45s] — End card**
- Bold: **"TezTekshir — oila bilan ishonch"**
