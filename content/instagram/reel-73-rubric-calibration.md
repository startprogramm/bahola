# Reel 73 — Two Teachers Calibrate a Shared Rubric (45 Seconds)

## Concept
Two teachers from the same school (or subject) use Bahola data to align their grading rubrics. Shows peer collaboration made possible by shared AI data. Core insight: when two teachers grade the same subject, Bahola data helps them spot inconsistencies and agree on standards. Bilingual — Uzbek + Russian.

## Hook (0–4s)
Split screen: two phones side by side.
Left phone shows Teacher A's assessment results. Right shows Teacher B's.
Bold overlay: **"Ikkita o'qituvchi, bitta sinf — nima farq bor?"** / **"Два учителя, один класс — где расхождение?"**

## Scene Breakdown

**[0–6s] — Muammo / Проблема**
- Split screen maintained
- Left: Teacher A's submission scores (avg 14/20)
- Right: Teacher B's scores (avg 11/20) — same students, same subject
- Overlay: "Bir sinf, bir fan — lekin natijalar farqli"
- Overlay (RU): "Один класс, один предмет — разные результаты"

**[6–16s] — Rubrikalarni solishtirish / Сравнение рубрик**
- Screen: both teachers open their mark schemes side by side on a tablet/computer
- Voiceover (UZ): "Biz rubrikalarimizni solishtirib ko'rdik — ikkita mezon farqli bo'lgan ekan"
- Cursor highlights two specific criteria that differ between rubrics
- Overlay: "📋 Farq topildi — 2 ta mezon"
- Voiceover (RU): "Нашли расхождение в 2 критериях"

**[16–28s] — Moslashtirishni ko'rsatish / Согласование**
- Screen: Teacher A edits their mark scheme — updates criterion wording to match
- Screen: both teachers' rubrics now match on those criteria
- Overlay: "✅ Birlashtirildi — endi standart bir xil"
- Voiceover (UZ): "Endi ikkalamiz ham bir xil mezonga ko'ra baholaymiz"
- Quick cut: re-run submission — score now consistent across both teachers

**[28–38s] — Natija / Результат**
- Screen: comparison of scores before/after calibration
- Bar chart overlay: scores converge from spread to tight cluster
- Overlay: "📊 Baholash izchilligi: +40%"
- Teacher A and B seen together, nodding at shared screen
- Voiceover (RU): "Теперь оценки честнее — для каждого ученика"

**[38–42s] — Xulosa / Вывод**
- Both teachers on camera together (brief)
- Overlay: "Hamkorlik — adolatli baholashning kaliti"
- Overlay (RU): "Совместная работа = справедливые оценки"

**[42–45s] — End card**
- Text: **"Bahola — maktabingiz uchun"**
- Bahola logo + "bahola.uz"

## Production Notes
- Two teachers should ideally be from the same school/subject (math, Uzbek language, etc.)
- Screen recordings: use real Bahola data — anonymize student names
- Split-screen segments: clean dividing line, both sides readable
- Voiceover alternates: Uzbek for steps 1-2, Russian for step 3 — shows bilingual community
- Subtitles recommended (both languages) — many viewers watch without sound
- Music: collaborative, calm — think "coworking" vibe, not dramatic
- End card: Bahola logo + "bahola.uz" + subscribe CTA
