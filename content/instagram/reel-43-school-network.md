## Concept

Visual journey showing TezTekshir spreading across schools in Uzbekistan. Animated map with school-by-school reveal, or fast montage of different school names/cities appearing. Tone: proud movement, growing community. Ends with a direct challenge to the viewer's school. Caption in Uzbek.

## Production Notes

- **Map style:** Clean, minimal. Dark background, bright accent color for dots (brand teal/blue). School names in white. Not a geographic precision tool — impressionistic is fine.
- **Animation:** Each city reveal should feel like a "notification" — quick, satisfying ping. Don't rush the early ones; let viewers read the first few. Speed up in the second half for momentum.
- **School names:** Use real school types (Prezident maktabi) where accurate. Don't name specific schools unless confirmed they've given permission.
- **"Sizning maktabingiz qachon?" hold:** This pause is the emotional hook — let it land. At least 1.5 seconds on this text.
- **Music:** Builds with the map. Should feel like a news ticker — urgent but positive. Arc: quiet intro → building momentum → peak → dramatic pause → warm close.
- **End card CTA:** "Bepul boshlang" + bio link. Keep it one action.
- **No faces needed:** This is a pure motion-graphics reel. If available, brief classroom footage can be used as map background at low opacity.
- **Duration:** Target 22–25 seconds. The reveal should feel fast enough to be exciting but not so fast viewers can't read the names.
