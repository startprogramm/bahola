## Production Notes

- **Key visual:** Animated score counter — satisfying number-go-up effect. Use motion graphics.
- **Authenticity:** Use real TezTekshir UI screenshots for feedback panel.
- **Color progression:** Red → Yellow → Orange → Green follows emotional arc.
- **Music:** Starts minimal/tense, builds to warm uplift. No lyrics. Think "study lo-fi meets triumph."
- **Pacing:** Each week transition should feel like a quick page flip — snappy, not slow.
- **Split screen:** Contrast should be clear but not exaggerated. Same actor, different body language.
