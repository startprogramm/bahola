# Reel 84 — Teacher Grades an Entire Class in Under 5 Minutes (Live Demo)

## Concept
Raw, real-time demonstration. A teacher sits down with a full class set of 25 student submissions — and the audience watches Bahola process them all in under 5 minutes. No talking head, no slideshow. Just the screen, a timer in the corner, and the satisfying sight of red "PENDING" statuses flipping to green "GRADED" one by one. The message is visceral: this is real, this is fast, and you're missing out if you're still grading by hand.

## Hook (0–5s)
Timer starts: **00:00**
Bold overlay (UZ): **"25 ta o'quvchi ishi. Tayyor bo'lishim uchun qancha vaqt kerak?"**
Overlay (RU): **"25 работ. Сколько мне нужно времени?"**
Teacher's finger hovers over "Ishlarni yuklash" button. Taps.

## Scene Breakdown

**[0–8s] — Yuklash / Загрузка**
- Teacher opens the assignment in Bahola
- Taps "Ishlarni yuklash" → device file picker opens
- Selects all 25 student work photos in one go (hold-select, bulk select visible)
- Hits "Yuborish" — upload progress bar appears
- Overlay (UZ): "25 ta rasm. Bir vaqtda."
- Overlay (RU): "25 фото. Одновременно."
- Timer: 00:47

**[8–20s] — AI baholayapti / AI проверяет**
- Submission list appears: 25 rows, all showing 🔄 PROCESSING
- First submission flips: 🔄 → ✅ BAHOLANDI — 87/100
- Second flips: ✅ BAHOLANDI — 74/100
- Third: ✅ 92/100
- Cascade of green checkmarks, accelerating
- Timer running in corner: 01:30 → 02:00 → 02:30
- Overlay (UZ): "AI har bir ishni baholash sxemasiga solishtiradi."
- Overlay (RU): "AI сверяет каждую работу с критериями оценивания."

**[20–35s] — Natijalar / Результаты**
- All 25 rows show ✅ BAHOLANDI — timer shows 03:52
- Teacher scrolls through: scores, AI feedback previews, one flagged "⚠️ Ko'rib chiqish tavsiya etiladi"
- Teacher taps the flagged work → reads AI note → adjusts score from 41 to 48
- Taps "Tasdiqlash"
- Overlay (UZ): "Shubhali ishlar avtomatik ajratib ko'rsatiladi."
- Overlay (RU): "Спорные работы выделяются автоматически."

**[35–45s] — Partiyaviy tasdiqlash / Пакетное подтверждение**
- Teacher returns to full list
- Taps "Hammasini tasdiqlash" — single tap
- All rows turn solid green with ✅ TASDIQLANDI
- Timer: 04:41
- Overlay (UZ): "25 ta ish. 1 ta tugma. Bajarildi."
- Overlay (RU): "25 работ. 1 кнопка. Готово."

**[45–50s] — End card**
- Final timer freeze: **04:41**
- Bold (UZ): **"4 daqiqa 41 soniya. 25 ta o'quvchi baholandi."**
- Bold (RU): **"4 минуты 41 секунда. 25 учеников проверено."**
- Bahola logo + "bahola.uz"

## Production Notes
- The **timer in the corner is non-negotiable** — it's the proof. Show it continuously from 00:00 to 04:41.
- Screen recording of real Bahola UI preferred. Anonymize student names (use initials or placeholder names).
- The cascade of ✅ BAHOLANDI statuses is the visual payoff — use a satisfying "pop" sound per flip or a subtle chime sequence.
- No voiceover needed — let the timer and overlays tell the story.
- Teacher's hands visible on phone/tablet — keeps it grounded and human.
- The flagged work moment (⚠️) is important — it shows the AI isn't just rubber-stamping, it's thinking. Keep it in.
- Audio: quiet focus ambiance → ticking clock urgency → satisfying completion jingle at the end.
