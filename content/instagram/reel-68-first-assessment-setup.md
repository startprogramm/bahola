# Reel 68 — First Assessment Setup (60 Seconds)

## Concept
Screen-capture style tutorial: teacher creates a class and first assessment from scratch in under 60 seconds. Fast-paced, step labels overlaid, upbeat music.

## Hook (0–3s)
Text on screen: **"Birinchi topshiriqni 60 soniyada yarating"**
Teacher hand taps phone screen — app opens.

## Scene Breakdown

**[0–10s] — Sinfni yarating**
- Screen: Dashboard → "Yangi sinf" button tap
- Overlay: "1-qadam: Sinf yarating"
- Teacher types class name: "10-A — Matematika"
- Quick tick animation: ✅ Sinf tayyor

**[10–25s] — Topshiriq yarating**
- Screen: Inside class → "Yangi topshiriq" tap
- Overlay: "2-qadam: Topshiriq qo'shing"
- Title: "1-chorak nazorat ishi"
- Total marks: 20
- Mark scheme: types 3 lines of criteria

**[25–45s] — Baholash sxemasini yuklang**
- Overlay: "3-qadam: Sxema yuklang (ixtiyoriy)"
- Drag-drop a PDF file onto the upload zone
- Progress bar fills → green checkmark
- Overlay: "AI sxemani o'qib oldi ✅"

**[45–55s] — Faollashtiring**
- Status toggle: DRAFT → ACTIVE
- Confetti burst animation
- Overlay: "O'quvchilar endi yuborishi mumkin!"

**[55–60s] — Result card**
- Split screen: before (stack of papers) vs after (TezTekshir screen)
- Text: **"5 daqiqadan kam vaqtda — tayyor!"**
