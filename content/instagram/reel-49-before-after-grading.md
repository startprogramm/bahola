## Concept

Split-screen "before and after" comparison showing a teacher's Friday evening — before TezTekshir vs after TezTekshir. Before: hunched over a pile of papers at 9pm, exhausted, red pen everywhere. After: phone on the table, results already done, teacher relaxed. Fast, punchy, immediately relatable. Tone: playful but true. Every teacher has lived the "before."

The message: you already know the before. This is the after.

## Production Notes

- **The core contrast is time.** 21:30 vs 18:45 is the story. Make the clocks prominent in both shots — they do most of the emotional work.
- **Same teacher, same day.** This is crucial. The "before" and "after" should feel like parallel universes — same person, same Friday, different tools. Use same wardrobe, same general energy/setting.
- **The BEFORE must be genuinely relatable** — not exaggerated to be cruel. Most teachers watching will recognize themselves immediately. Stack of papers, red pen, that specific tired posture. This is a loving tribute to "the old way," not mockery.
- **The AFTER must feel earned, not smug.** No "look how easy" energy. Just: "this is actually real and you can have it too."
- **Split-screen execution options:**
  - Option A: Literal split-screen — shoot both scenes and composite side-by-side.
  - Option B: Sequential "before → after" with a visual wipe transition.
  - Option C: Fast cuts alternating between before and after, then resolution.
  - Option B is simplest and most effective for Instagram format.
- **Clock detail:** The specific times (21:30 vs 18:45) are the punchline. Make them big and visible. If using a real clock, set them. If using screen overlay text, make the font bold.
- **Shot 4 (detail comparison):** These quick cuts are the "proof" moment. Show real TezTekshir UI on the "after" side — actual feedback text, actual score dashboard. Not a mockup.
- **Music:** Start slightly heavy/tired for "before," then a clean beat drop into "after." Should feel like tension → release. Classic before/after audio storytelling.
- **Duration:** Target 30–40 seconds. This concept works best when punchy. Don't over-explain.
- **No voiceover needed.** The visuals + text overlays carry it.
