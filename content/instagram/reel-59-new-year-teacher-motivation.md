# Reel 59 — New Year Teacher Motivation: 2026 → 2027

**Theme:** Reflect on how far teachers have come with Bahola; inspire them for 2027

**Text overlay:** "2026 yil boshlanganda..."

## Production Notes
- Split-screen effect is the visual backbone — keep it consistent
- Paper stack side: intentionally chaotic, dim lighting
- Dashboard side: bright, clean, organized
- Teacher should be the same person in both shots for contrast
- Stats animation should match the teacher's real semester if possible (or use believable mockup numbers)
