# Reel 69 — Student Join Flow (30 Seconds)

## Concept
Split-screen: left side shows teacher sharing the invite code; right side shows student joining from their device. Fast, clean, satisfying.

## Hook (0–3s)
Bold text center screen: **"O'quvchilaringiz 30 soniyada qo'shilsin"**
Both phones shown side by side.

## Scene Breakdown

**[0–8s] — Teacher shares code**
- Left screen: Teacher in class → Members tab → invite code visible: "KL7-MN4"
- Teacher taps "Kodni nusxalash" (Copy code)
- Overlay (left): "👨‍🏫 O'qituvchi: kodni ulashing"
- Quick zoom on code

**[8–18s] — Student enters code**
- Right screen: Student opens Bahola → "Sinfga qo'shilish" tap
- Types invite code: K-L-7-M-N-4
- Taps "Qo'shilish" button
- Overlay (right): "👩‍🎓 O'quvchi: kodni kiriting"

**[18–26s] — Student sees class stream**
- Right screen: Student lands in class stream — class name banner, welcome message, first assignment visible
- Green success badge: "✅ Sinfga muvaffaqiyatli qo'shildingiz!"
- Left screen: Teacher's Members tab — new student name appears with animated pop-in

**[26–30s] — Result**
- Both screens shown side by side
- Overlay: **"Tayyor! 30 soniya."**
- Bahola logo bottom center
