# Reel 87 — "End-of-Quarter Report Card in 60 Seconds"

## Concept
Fast, satisfying screen-capture demo. A teacher closes out a quarter and generates a full class report card — all scores, all students, all subjects — in under 60 seconds. No Excel. No copy-paste. No late nights. The reel is pure product proof: show it, don't tell it. Target: teachers and directors who dread end-of-quarter reporting.

## Hook (0–4s)
Text appears word by word, punchy:
Bold (UZ): **"Chorak tugadi. Tabellarga 3 kun ketadimi?"**
Bold (RU): **"Четверть закончилась. Три дня на табели?"**
Cut hard to: Bahola dashboard. Timer starts in corner: 00:00.

## Scene Breakdown

**[0–4s] — Hook**
- Black background, white text countdown feel
- UZ: "Chorak tugadi."
- RU: "Четверть закончилась."
- UZ: "Tabellarga 3 kun ketadimi?"
- RU: "Три дня на табели?"

**[4–14s] — Dashboard navigatsiyasi / Навигация по дашборду**

- Teacher opens Bahola → navigates to "Chorak hisoboti / Отчёт за четверть"
- Screen shows: class dropdown → select class → select quarter
- Student list populates automatically — all names, all scores, color-coded
- Timer corner: 00:12

Screen overlays (UZ): "Sinfni tanlang → Chorakni belgilang → Tayyor."
Screen overlays (RU): "Выберите класс → Укажите четверть → Готово."

**[14–24s] — Hisobot ko'rinishi / Просмотр отчёта**

- Full class report card on screen:
  - Student names (blurred for privacy)
  - Subject columns: Math / Physics / Chemistry
  - Scores per assessment visible
  - Quarter average auto-calculated per student
  - Color coding: green = pass, yellow = borderline, red = failing
- Teacher scrolls through — 28 students visible in seconds
- Taps "PDF yuklab olish / Скачать PDF" → file downloads instantly

Timer corner: 00:21

Overlay (UZ): "28 ta o'quvchi. Barcha fanlar. 21 soniyada."
Overlay (RU): "28 учеников. Все предметы. За 21 секунду."

**[24–28s] — Ulashish / Поделиться**

- Teacher taps "Direktorga yuborish / Отправить директору"
- Share dialog: Telegram icon selected → sent
- Director's phone notification pops: "Yangi hisobot keldi ✅"

Overlay (UZ): "Bir tapping bilan — direktorga."
Overlay (RU): "Одним нажатием — директору."

**[28–30s] — End card**
- Timer stops: 00:29
- Bold (UZ): **"3 kun → 29 soniya."**
- Bold (RU): **"3 дня → 29 секунд."**
- Bahola logo + "bahola.uz"

## Production Notes
- Screen recording must be clean and unhurried — each tap clearly visible, no fumbling
- Timer in corner (00:00 counting up) is the hook's payoff — must be visible throughout
- Student names MUST be blurred — use placeholder names or pixel blocks. No real student data.
- The "3 kun → 29 soniya" end card stat is the punchline — give it 2 full seconds on screen
- PDF download animation should feel satisfying — brief loading arc, then ✅ appears
- Aspect ratio: 9:16 (Reels)
- Duration: 28–32 seconds
- Music: crisp, efficient, slightly techy — something that conveys speed without anxiety
