📝 **Какой тип задания выбрать? Объясняем просто**

В TezTekshir поддерживаются разные форматы заданий. Вот когда использовать каждый из них:

**Открытый ответ (эссе / развёрнутый ответ)**
→ Когда важен процесс рассуждения, аргументация, личная позиция
→ Примеры: сочинение, анализ текста, объяснение решения задачи
→ ИИ оценивает структуру, полноту, логику — по вашей шкале

**Структурированный ответ**
→ Когда нужно проверить конкретные шаги или факты
→ Примеры: математическое решение, химическое уравнение, список дат
→ ИИ сравнивает каждый элемент с эталоном и даёт баллы за каждый шаг

**Смешанный формат**
→ Когда задание состоит из нескольких частей разного типа
→ Например: 2 коротких вопроса + 1 развёрнутый ответ
→ Задайте разные критерии для каждой части — ИИ справится

💡 **Совет:** Начните с простого структурированного задания, чтобы увидеть систему в действии. Потом переходите к эссе.
